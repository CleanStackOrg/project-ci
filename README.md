New Build Pipeline System for Continuous Integration

- pipeline (stages, dsl)
- plan (trello, burndown)
- specs (cucumber editor)
- code (git, svn, cvs)
- build (maven, gradle)
- doc (wiki, site)
- deliver (nexus, repo)
- deploy (ansible)
- run (docker)
- report (cucumber results, bui)
- qa (sonar)
